# Внешность персонажей

Сюда складываются текстовые портреты персонажей.

Лучше хранить несколько версий, если внешний вид еще не утвержден:

```text
alexandros_vneshnost_v01.md
alexandros_vneshnost_v02.md
mihael_vneshnost_v01.md
```

## Что фиксировать

- возраст на вид;
- лицо, волосы, глаза, осанка;
- одежда и доспехи;
- символы власти;
- манера речи и жесты;
- первое впечатление.
