# География текстом

Сюда складываются текстовые описания мира, Империи, регионов и маршрутов.

Для каждой версии географии желательно делать короткую заметку:

```text
Название:
Версия:
Дата:
Что в описании канон:
Что пока примерное:
Связанные главы:
```

## Что описывать

- север, юг, запад, восток;
- соседние регионы;
- дороги и торговые пути;
- горы, леса, реки, моря;
- опасные зоны;
- расстояния качественно: близко, далеко, несколько дней пути, долгий поход.
