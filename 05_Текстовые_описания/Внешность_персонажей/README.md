# Внешность персонажей

Сюда складываются текстовые портреты персонажей. Основные изображения персонажей лежат отдельно в `11_Медиа/Портреты_персонажей`.

Лучше хранить несколько версий, если внешний вид еще не утвержден:

```text
alexandros_vneshnost_v01.md
alexandros_vneshnost_v02.md
mihael_vneshnost_v01.md
```

## Что фиксировать

- возраст на вид;
- лицо, волосы, глаза, осанка;
- одежда и доспехи;
- символы власти;
- манера речи и жесты;
- первое впечатление.

## Связь с изображениями

- Если портрет уже есть, текстовое описание должно учитывать его как визуальный референс.
- Если текст и изображение расходятся, приоритет у утвержденного текста, пока пользователь явно не утвердит новый визуальный канон.
- Если у персонажа нет портрета, текстовое описание можно использовать как основу для генерации.
