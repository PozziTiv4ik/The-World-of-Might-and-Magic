Эта папка хранит основные портреты персонажей и файлы промптов портретов.

## Структура

Основной портрет персонажа хранится так:

```text
11_Медиа/Портреты_персонажей/Имя_Персонажа/Имя_Персонажа_основной_портрет.png
```

Рядом хранится промпт:

```text
11_Медиа/Портреты_персонажей/Имя_Персонажа/Промпт_портрета.md
```

Главный индекс портретов:

```text
11_Медиа/Портреты_персонажей/Индекс_портретов.md
```

## Рабочие правила

Все правила генерации, формата 3 на 4, референсов, сохранения файла и обновления карточек вынесены в:

```text
00_Инструкции_для_ИИ/02_Портреты_персонажей.md
```

Не добавляй сюда новую агентскую логику. Если правило меняется, обновляй файл в `00_Инструкции_для_ИИ`.
