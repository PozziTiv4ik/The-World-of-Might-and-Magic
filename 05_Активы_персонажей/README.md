Эта папка хранит карточки активов персонажей: войск, личных отрядов, флотских частей, предметов, артефактов, владений и других сущностей, которые связаны с конкретным владельцем.

## Структура

```text
05_Активы_персонажей/00_Индекс_активов.md
05_Активы_персонажей/Имя_Владельца/00_Индекс_активов.md
05_Активы_персонажей/Имя_Владельца/Название_актива.md
```

## Рабочие правила

Правила создания, обновления и визуального сопровождения активов лежат в:

```text
00_Инструкции_для_ИИ/05_Активы_персонажей.md
```

Не веди здесь сценовые иллюстрации и карты. Визуалы активов относятся к самим отрядам, предметам или экипировке.
