# Текстовые описания

Эта папка нужна для визуальных описаний словами: внешность, атмосфера, география, предметы и сцены. Реальные изображения и портреты теперь хранятся отдельно в `11_Медиа`.

## Рекомендуемая структура

- `География_текстом` - карта мира словами: соседство, дороги, моря, горы, маршруты.
- `Внешность_персонажей` - портреты словами: одежда, возраст, манера держаться, символы.
- `Атмосфера_сцен` - описание важных сцен: место, настроение, свет, звуки, детали.

## Принцип

Текстовые описания фиксируют смысл и канон. Изображения из `11_Медиа` используются как визуальные референсы, но не переписывают текстовый канон без отдельного подтверждения.

## Как называть файлы

Лучше использовать понятные имена:

```text
imperiya_geografiya_v01.md
alexandros_vneshnost_v01.md
belaya_gavan_priem_evanessy_atmosfera_v01.md
```

Версии полезны, потому что описание мира может уточняться.
