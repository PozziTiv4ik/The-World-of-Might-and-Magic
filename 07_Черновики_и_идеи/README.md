# Черновики и идеи

Эта папка нужна для всего, что еще не стало каноном:

- новые сообщения из чата;
- идеи игроков;
- варианты будущих сцен;
- спорные изменения;
- заготовки для нейронки;
- реткон-заявки.

## Важное правило

Черновик не считается каноном, пока не перенесен в кампанию, лор, персонажа, локацию или архив.
