# Архив канона

Здесь хранятся завершенные главы и события, которые больше нельзя менять по смыслу.

## Правило

Если глава попала сюда, она становится каноном.

Можно исправлять опечатки и формат, но нельзя менять смысл решений, событий и последствий.

## Куда класть завершенные главы

`06_Архив_канона/Завершенные_главы`

## Что добавлять в конец завершенной главы

```text
Итог:
Главные решения:
Последствия:
Открытые крючки:
Связанные персонажи:
Связанные локации:
```
