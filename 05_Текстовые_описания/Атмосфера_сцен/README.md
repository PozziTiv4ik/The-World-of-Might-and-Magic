# Атмосфера сцен

Сюда складываются текстовые описания событий, городов, битв, предметов и атмосферных моментов.

Пример:

```text
glava_01_doklad_dariusa_atmosfera_v01.md
glava_01_sdelka_uaitburga_atmosfera_v01.md
```

## Что описывать

- место;
- погоду и время суток;
- свет, звук, запахи;
- настроение сцены;
- важные предметы;
- детали, которые помогают потом продолжать повествование.
