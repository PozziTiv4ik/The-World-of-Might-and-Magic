Эта папка хранит визуальные материалы игры: портреты персонажей, будущие иллюстрации сцен, референсы стиля и служебные правила генерации.

Текстовый канон остается главным. Изображение помогает держать лицо, костюм, настроение и визуальный стиль, но не переписывает биографию или событие само по себе. Если картинка противоречит карточке персонажа, лору или сцене, приоритет у текста, пока пользователь явно не утвердит изменение.

## Структура

- `Портреты_персонажей` - основные портреты, варианты и референсы персонажей.
- `Карты_и_схемы` - карты, торговые схемы, маршруты и визуальные пометки опасных зон.

## Общие правила

- Пользователь не обязан раскладывать файлы вручную: ассистент сам переносит, переименовывает и обновляет ссылки.
- Для значимых персонажей нужно стремиться к наличию основного портрета.
- Новые изображения не создаются автоматически без согласия пользователя, но ассистент регулярно предлагает подходящую тему для генерации.
- Все файлы с портретами должны иметь понятные канонические названия, без случайных временных имен.
